# GitHub Pages starter for the cubo EO tutorial

This folder is ready to use as a simple GitHub Pages site.

## How to publish

1. Create a GitHub repository and upload everything in this folder.
2. On GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the branch you pushed (for example `main`) and choose the **`/docs`** folder.
5. Save.

GitHub Pages will publish the site from the `docs/` folder.
