# CEOS UNOOSA Training Material

This repository supports a hands-on Earth Observation training activity focused on climate resilience and risk monitoring.

It contains the teaching notebooks, slide materials, and documentation pages used during the session, and serves as the central hub for participants before, during, and after the training.
